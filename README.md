# SQL-Fundamentals-exercises
Purpose:
This repository documents the SQL fundamentals exercises I’ve completed as part of my data analytics training. It serves as a personal record of my progress, showcasing the concepts I’ve practiced and the queries I’ve written

Topics covered:
Introduction to Databases – Tables, rows, columns, and relationships
Introduction to SQL – Basic query structure and syntax
Filtering Statements – Using WHERE to refine results
SQL Operators – Arithmetic, logical, and comparison operators
Sorting Statements – Ordering data with ORDER BY
Aggregate Functions – Summarizing data with COUNT, SUM, AVG, MAX, MIN
GROUP BY Statements – Grouping data for analysis
CASE Statements – Conditional logic in queries
JOIN Statements – Combining data across multiple tables

How I use this Repo:
Store sql files for each exercise
Keep datasets or examples linked to the queries
Track my learning progress step by step
Revisit solutions to reinforce concepts
